# 🛒 Klif Supermarket Scanner

Aplicativo web progressivo (PWA) de uso pessoal desenvolvido para facilitar o registro de compras e preços de supermercado diretamente pelo celular, com leitura de código de barras em tempo real.

## ✨ Recursos Principais:
- 📴 **100% Offline First:** Todo o banco de dados de produtos e histórico de listas de compras é mantido localmente utilizando IndexedDB.
- 🔒 **Acesso Restrito:** Tela de bloqueio (lock screen) segura com validação criptográfica SHA-256 local no navegador (sem tráfego de senhas pela rede).
- 📸 **Leitor de Código de Barras (EAN):** Leitura de alta performance direto da câmera traseira de smartphones ou entrada manual rápida.
- ☁️ **Integração com Open Food Facts API:** Busca automática opcional de nome, marca e fotos de produtos novos na nuvem.
- 📱 **Instalação PWA Premium:** Otimizado com efeitos de glassmorphism e suporte total à instalação de atalho de tela no iOS (Safari) e Android (Chrome).
